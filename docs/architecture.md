# アーキテクチャとプロジェクト構成

## アーキテクチャ

- **メインソルバー**: `@src/main.rs` - コアとなる最適化アルゴリズムが含まれる。
- **ツール連携**: 自動テスト用の `pahcer` ランナーと、可視化用の `pahcer-studio` を使用している。

## プロジェクト構成

```
src/main.rs             # ランダム探索アルゴリズムを含むメインソルバー
tools/                  # コンテスト用ツール（生成器、可視化器） **変更禁止**
  ├── in/               # 入力テストファイル
  ├── out/              # 出力ファイル
  └── src/              # ツールバイナリ (gen, vis)
pahcer/                 # テスト実行結果 **変更禁止**
pahcer-studio/          # ウェブベースの可視化ツール (TypeScript/Vite) **変更禁止**
run.sh                  # 単一テスト実行スクリプト
PROBLEM.md              # Markdown 形式の問題文 **変更禁止**
PROBLEM.html            # HTML 形式の問題文 **変更禁止**
LOG.md                  # 解法アプローチの解説と記録
optimize.py             # Optuna を用いたパラメータ最適化スクリプト
```
